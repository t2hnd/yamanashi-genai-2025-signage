# デジタルサイネージシステム 開発仕様書
# 〜 デモ対応版 〜

## プロジェクト概要

### 基本情報
- **プロジェクト名**: 北杜市パン屋デジタルサイネージ（デモ版）
- **目的**: 利益率の高い商品への誘導、併売促進、在庫最適化による売上・利益向上
- **設置場所**: 店内（レジ横、商品棚前）
- **技術スタック**: React + TypeScript + Tailwind CSS

### ビジネス背景
- 山梨県北杜市（八ヶ岳山麓）の観光地パン屋
- 夏季観光シーズンは売上1.6倍
- 客単価: 現状¥187 → 目標¥200

#### 2.2 デモシナリオ

**シナリオ2: 在庫切れ対応**
- 人気商品の在庫を徐々に減少
- 在庫切れ時に代替商品（高利益率）への切替を表示

**シナリオ3: 季節比較**
- 夏（観光客向け）と冬（地元向け）を切り替えて表示変化を確認

## 推奨ロジック

### スコアリング関数
```typescript
function calculateScore(
  product: Product,
  timeSlot: TimeSlot,
  season: SeasonConfig,
  inventory: InventoryItem,
  weights: ScoreWeights
): Recommendation {
  // 各スコアを0-100で計算
  const profitScore = product.profitMargin;
  const timeSlotScore = calculateTimeSlotScore(product, timeSlot);
  const seasonScore = calculateSeasonScore(product, season);
  const inventoryScore = calculateInventoryScore(inventory);
  
  // 重み付け合計
  const totalScore = 
    profitScore * weights.profit +
    timeSlotScore * weights.timeSlot +
    seasonScore * weights.season +
    inventoryScore * weights.inventory;
  
  return {
    product,
    score: totalScore,
    scoreBreakdown: { profitScore, timeSlotScore, seasonScore, inventoryScore },
    reason: generateReason(...)
  };
}
```

### 在庫スコア計算
```typescript
function calculateInventoryScore(inventory: InventoryItem): number {
  if (inventory.status === 'out') return 0;       // 在庫切れ
  if (inventory.status === 'low') return 50;      // 在庫少（売り切りたい）
  
  const ratio = inventory.quantity / inventory.maxQuantity;
  if (ratio > 0.8) return 90;   // 在庫過多（積極的に推奨）
  if (ratio > 0.5) return 100;  // 適正在庫
  return 70;                     // やや少なめ
}
```
